# laravel-note

Laravel Note

-> Valet Packages

- Install laravel valet packages to composer
  - composer global require laravel/valet
- install valet to local
  - valet install [project-name]
- change version valet php

  - valet use php@version (php@latest) / php@8.1.10
  - valet use php

-> Breeze (authentication like jetstream but more simple)

- composer require laravel/breeze --dev
- php artisan breeze:install

- show all link
  - valet links
- link app to valet
  - valet link [name-app]

-> Migration

- php artisan migrate
- php artisan migrate:rollback
- php artisan migrate:fresh
- php artisan migrate:fresh --seed

<!-- First Method -->

- php artisan make:migration [name-migration] --table=[name-table]
- php artisan make:model [name-model]

<!-- Second Method : include migration-->

- php artisan make:model [name-model] -m

-> Seeder

- php artisan make:seeder [name-seeder]
- register ke DatabaseSeeder
- php artisan db:seed
